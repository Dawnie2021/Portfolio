Work

The first challenge in my coding bootcamp was focused on semantic code refactoring. The challenge involved taking poorly written, non-semantic HTML code and refactoring it to make it more structured, accessible, and easier to maintain. I quickly realized that the key to semantic code is clear and concise HTML tags for different types of content. I began by analyzing each section of the HTML snippet and adding proper semantic tags to accurately describe its content. To improve accessibility, I added alt attributes to all images and included a descriptive title element in the head of the document. I also ensured that all links had descriptive anchor text that explained their purpose and destination. Throughout the challenge, I remained mindful of the importance of clean and readable code. By the end of the challenge, I had transformed a jumbled, poorly written HTML page into a well-structured and readable document. Overall, the experience helped me to understand the importance of clear, semantic, and accessible code when developing web applications.
